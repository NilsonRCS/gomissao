# gomissao

Sistema web em Go para cálculos de comissões, controle de estoque e juros.

## Como rodar

1. Clone o repositório:
```bash
git clone https://github.com/NilsonRCS/gomissao.git
cd gomissao
```

2. Execute o servidor:
```bash
go run main.go
```

3. Acesse no navegador:
```
http://localhost:5555
```

## Desafios implementados

- **Desafio 1**: Cálculo de comissões de vendedores
- **Desafio 2**: Controle de estoque com movimentações
- **Desafio 3**: Cálculo de juros sobre valores vencidos
